# Adobe Claude MCP Automation

A portfolio project focused on connecting Claude with Adobe Photoshop and Adobe InDesign through MCP server automation.

This project is based on customizing existing open-source MCP server workflows. I adapted the setup for local Adobe automation, fixed integration issues, and tested Claude-assisted workflows with Photoshop and InDesign.

## My Contribution

- Configured local MCP server setup for Claude Desktop Adobe Photoshop, Adobe Illustrator and Adobe Acrobat   workflows
- Fixed setup/integration issues
- Tested local automation through Claude
- Prepared example configuration without secrets
